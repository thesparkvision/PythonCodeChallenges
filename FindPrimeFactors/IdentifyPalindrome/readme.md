## Question
Write a python function to determine if a given string is a palindrome

### Input
String

### Output
Boolean value

### Constraints
- Only consider letters (A-Z)
- Ignore case (for example, 'A' == 'a')

### Test Cases
> is_palindrome("racecar") 
  => True

> is_palindrome("a")
  => True

> is_palindrome("Toy")
  => False
  
> is_palindrome("Racecar")
  => True
